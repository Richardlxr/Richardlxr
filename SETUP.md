# Richardlxr Profile README Setup

This folder is designed to become the special GitHub profile repository:

```text
Richardlxr/Richardlxr
```

When the repository is public and contains `README.md` at the root, GitHub displays
that README on `https://github.com/Richardlxr`.

## Publish

1. Create a public GitHub repository named `Richardlxr`.
2. Push this folder to that repository.
3. Open the repository's Actions tab and allow workflows if GitHub asks.
4. Run `Generate contribution snake` once, or wait for the scheduled run.

The snake animation is generated into the `output` branch. Until that workflow runs,
the contribution animation section may appear blank.

## Local commands

```bash
git init
git add README.md SETUP.md assets .github
git commit -m "Create profile README"
git branch -M main
git remote add origin git@github.com:Richardlxr/Richardlxr.git
git push -u origin main
```
